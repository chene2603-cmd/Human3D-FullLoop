# 渗透·对抗者模块（Penetration / Adversary）

> ⚠️ **本模块仅用于非商业学术研究，严格禁止用于攻击第三方系统或服务。**

## 🎯 目标

测试 **本地部署的 3D 人体重建模型**（如 SMPL-X、Gaussian Avatars）在面对对抗扰动时的鲁棒性。

典型应用场景：
- 在实验室环境中评估自研模型的抗干扰能力
- 生成对抗样本用于后续防御训练（见 `defense/` 模块）
- 学术论文中的消融实验（ablation study）

## 🧪 当前支持方法

- **FGSM (Fast Gradient Sign Method)**  
  对输入关节点坐标（如 `[N, J, 3]`）添加微小扰动，最大化预测误差。

> 💡 输入可为：2D/3D 关键点、网格顶点、NeRF 参数等——取决于你的本地模型接口。
