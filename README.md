# Human3D-FullLoop 项目路线图

本项目采用 **渐进式开源策略**，核心目标：  
✅ 推动 3D 人体重建学术研究  
⚠️ 严格限制非授权商业使用  

---

## 🟢 已开源模块（可自由使用，遵守 LICENSE）

- **哨兵·预言者 (`sentinel/`)**
  - 实时追踪 arXiv 论文
  - 关键词驱动热点预测
  - 输出优先级论文列表

---

## 🟡 计划开源模块（需安全审查后发布）

- **渗透·对抗者 (`penetration/`)**
  - 模拟对抗攻击（FGSM, PGD）
  - 仅支持本地模型测试
  - ❌ 不包含对第三方服务的真实攻击逻辑

- **认知免疫反哺 (`defense/`)**
  - 基于漏洞报告生成对抗样本
  - 触发模型在线微调（LoRA）
  - 依赖本地训练环境

---

## 🔴 永不公开模块（仅作者私有部署）

- **地图·蜜罐 (`honeypot/`)**
  - 边缘 Wi-Fi 蜜罐探测
  - 学术会议现场行为分析
  - ⚠️ 因涉及网络合规风险，**代码不公开**
