## 🧩 Core Modules

- **Sentinel (哨兵·预言者)**: 实时追踪 arXiv 最新 3D 人体重建论文，智能预测研究热点。
- **Adversary (渗透·对抗者)**: 自动对公开 Demo 发起对抗攻击，评估模型鲁棒性。
- **Honeypot (地图·诱饵者)**: 部署边缘 Wi-Fi 蜜罐，探测学术会议现场设备行为（仅限授权环境）。
- **Defense (认知免疫反哺)**: 基于漏洞报告自动生成对抗样本，触发模型在线微调。
