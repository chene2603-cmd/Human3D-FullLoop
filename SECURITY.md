# 安全策略（Security Policy）

## 🛡️ 项目定位

Human3D-FullLoop 是一个 **非商业学术研究项目**，所有模块设计均遵循：
- **最小权限原则**（无网络请求、无远程执行）
- **本地闭环原则**（所有操作需用户主动在本地运行）
- **伦理优先原则**（禁止用于攻击、监控或商业牟利）

## 🚫 禁止行为

以下行为明确违反本项目 LICENSE 和道德准则：
- 利用本代码攻击第三方 API、网站或服务
- 将对抗样本用于未授权的系统测试
- 绕过任何系统的安全防护机制
- 用于生成虚假身份、深度伪造等违法用途
